# GovAssist AI Backend

Backend API for GovAssist AI, an AI-powered Government Scheme Advisor.

## Tech Stack
- FastAPI
- LangChain
- FAISS
- Groq LLM
- Hugging Face Embeddings

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python run.py
```

## Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
```
